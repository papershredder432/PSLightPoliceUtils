# PSLightPoliceUtils
RocketMod plugin to police players. (Tase/Cuff)

Permissions:
`ps.lightpoliceutils.cuff`
`ps.lightpoliceutils.uncuff`
`ps.lightpoliceutils.tase`,
`ps.lightpoliceutils.untase`

Commands:
`/cuff <Player>`
`/uncuff <Player>`,
`/tase <Player>`,
`/untase <Player>`
